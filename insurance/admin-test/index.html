사양 검토용 목업 v0.1

[빌리지점] 스마트 업무 처리 원스톱 보드

구글폼 한 장으로 받던 배서 업무를, 업무구분에 맞는 정보만 정확히 받고 · 처리 상태를 실시간 공유하는 보드로 재구성한 목업입니다.

이 화면은 사내 전산팀·AI 개발자가 풀스택으로 구현할 수 있도록 만든 인터랙티브 사양서입니다. 모든 데이터는 가공된 가짜 데이터이며, 어떤 정보도 외부로 전송되지 않습니다.

역할별로 미리 보기

각 카드를 클릭하면 해당 역할로 로그인된 상태로 페이지를 보여드립니다. 화면 우상단 드롭다운으로 언제든 역할을 바꿔볼 수 있습니다.

이 목업이 보여주는 것

  • 업무구분 7종에 따라 필수 입력 필드가 동적으로 바뀌어, 카드변경인데 카드번호가 없는 누락이 시스템적으로 차단됩니다.
  • 상담사는 "내 요청" 마이페이지에서 자기 제출 건의 상태(접수·진행중·완료·반려·보류)를 한 곳에서 봅니다. "처리 회신이 없다"는 불만 해소.
  • 총무는 처리 보드에서 필터·인라인 상태 변경·"내가 잡기"로 빠르게 일을 진행합니다.
  • 상태가 바뀌면 상담사에게 자동 알림이 갑니다 (실제 구현 시 이메일·카카오 알림톡).
  • 관리자는 통계 화면에서 처리량·평균 소요시간·반려율을 봅니다.

실제 구현 방향

이 목업은 정적 HTML + localStorage로 동작하지만, 실제 운영은 두 가지 옵션으로 갈 수 있습니다:

  • 옵션 A — Google Apps Script + Sheets (단기 대체 권장): 기존 구글폼·시트 워크플로와 마찰 0, 무료, 1주 안에 MVP 가능
  • 옵션 B — Next.js + Supabase: 정식 시스템화. 동시 사용자·통계 분석 확장에 적합

자세한 사양은 SPEC.md 문서를 참고하세요. 이 문서를 AI 개발 어시스턴트에게 그대로 전달하면 즉시 풀스택 구현을 시작할 수 있습니다.
