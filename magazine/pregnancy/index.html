PREGNANCY

임신 중 태아보험,
이야기로 정리해드려요.

가입 골든타임(12–22주), 1·2차 기형아 검사 결과별 시나리오,
양수검사·NT 검사 영향, 거절·재가입 케이스를 모아두었어요.

발행 글

5편 · 매주 2~3편 추가 예정

  • 가이드

    1차 기형아 검사 결과가 영향을 주는 가입 조건

    임신 11~13주에 진행하는 1차 기형아 검사(NT·혈액 통합 평가)의 결과가 태아보험 청약 심사에 어떻게 반영되는지, 위험도 분류별 가입 시나리오를 정리해드려요.

  • 가이드

    NT 검사 결과 두께별 가입 시나리오 — 정상·경계·두꺼움 케이스 정리

    NT(목투명대) 검사 결과는 태아보험 가입 가능성과 부담보 조건에 직접 영향을 줘요. 두께별로 어떤 시나리오가 펼쳐지는지, 추가 검사 권유받은 경우 어떻게 진행할지 정리해드려요.

  • 가이드

    임신성 당뇨·고혈압 진단받은 산모의 태아보험 선택지

    임신성 당뇨·임신성 고혈압·자간전증 같은 합병증 진단을 받으셨다면 가입 가능 보험사와 보장 조건이 달라져요. 진단 시점·중증도별 시나리오와 신생아 보장에 미치는 영향을 정리해드려요.

  • 가이드

    양수검사·융모막검사 후에도 태아보험 가입할 수 있나요

    양수검사나 융모막검사를 받으셨거나 받기로 결정하셨다면 보험 가입 시점이 달라져요. 검사 결과가 정상·이상 소견·미정인 경우의 가입 가능성과 보험사 선택지를 정리해드려요.

  • 가이드

    태아보험 가입 골든타임 — 임신 12주에서 22주 사이에 결정해야 하는 이유

    태아보험은 임신 주차에 따라 가입 가능 여부와 보장 범위가 달라져요. 12주에서 22주 사이를 가입 골든타임이라 부르는 이유와, 시점을 놓쳤을 때 가능한 선택지를 정리해드려요.
