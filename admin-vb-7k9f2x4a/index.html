관리 TIPS

📍 Search Console 색인 요청 항상 3종 셋트: Google + Naver Search Advisor + Bing Webmaster. 한 곳만 하면 트래픽 절반 손해.

✅ 새 페이지 배포 후 체크리스트: ① sitemap.xml 갱신 → ② rss.xml 갱신 → ③ Pillar/허브 내부 링크 → ④ 3종 색인 요청.

⚠️ 카톡 OG 캐시: 1~3일 캐싱됨. 빠른 갱신은 Kakao 디버거에서 URL 캐시 삭제.

🔀 PR 워크플로우: git checkout -b feature/이름 → 작업 → push → PR 생성. 검토 후 머지.

🎯 Cluster 전략: 새 카테고리는 Pillar 1 + Spokes 5~7 형태로. 양방향 인바운드 링크가 토픽 권위도의 핵심.
