임산부 인증 — 한 번만 등록

임산부 할인은 한 번 인증을 등록해 두면 이후 결제 시 자동으로 적용돼요. 매번 인증을 반복할 필요가 없어요.

  • 1단계 · 임신확인서 발급 — 산부인과에서 임신확인서 발급
  • 2단계 · 앱 다운로드 — 코레일톡 또는 SRT 앱 설치
  • 3단계 · 임산부 인증 신청 — 앱 내 멤버십·할인 → 임산부 인증 선택
  • 4단계 · 서류 업로드 — 임신확인서 또는 산모수첩 사진 + 본인 인증
  • 5단계 · 인증 완료 — 보통 1~3영업일 내 인증 완료, 이후 자동 할인

인증 후 자동 적용 — 한 번 인증하면 출산 후 1년까지 결제 시 자동으로 할인 가격이 표시돼요. 매번 임신확인서를 다시 제출할 필요 없어요.

좌석 업그레이드 — 일반실 가격으로 특실

임산부 할인에서 가장 인기 있는 혜택은 좌석 업그레이드예요. 일반실 운임으로 결제하면 특실로 자동 업그레이드되는 방식이에요.

  • 적용 조건 — 임산부 인증 완료 + 해당 열차 특실 좌석 여유
  • 업그레이드 시점 — 결제 단계에서 자동 적용 (별도 신청 불필요)
  • 동반 1인 — 같은 결제 건에 포함하면 동반자도 업그레이드
  • 제한 사항 — 좌석 여유가 없으면 일반실 그대로 (할인은 그대로 적용)

특실은 좌석 간격이 넓고 등받이 각도가 더 편안해서 임신 후기 장거리 이동에 큰 도움이 돼요. 일반실 가격으로 특실 좌석을 받는 셈이라 가성비가 매우 높아요.

자주 묻는 질문

임산부 KTX 할인은 어떻게 받나요?

코레일 홈페이지나 앱에서 임산부 인증을 한 번 등록하면 이후 결제 시 자동으로 할인이 적용돼요. 인증을 위해서는 임신확인서 또는 산모수첩 사진과 본인 인증이 필요해요. 한 번 등록하면 출산 후 1년까지 할인이 유지돼요.

  • 등록 채널 — 코레일톡 앱 또는 코레일 홈페이지
  • 필요 서류 — 임신확인서·산모수첩 사진
  • 유지 기간 — 출산 후 1년

임산부 동반 가족도 같이 할인되나요?

임산부 본인은 운임 할인을 받고, 동반자 1인은 좌석을 일반실에서 특실로 무료 업그레이드받을 수 있어요. 다만 업그레이드는 좌석 여유가 있을 때만 가능해요.

  • 본인 혜택 — 운임 할인 + 좌석 업그레이드
  • 동반 1인 혜택 — 좌석 업그레이드만 적용
  • 업그레이드 조건 — 해당 열차 특실 좌석 여유 시

SRT도 동일한 혜택을 제공하나요?

네, SRT도 코레일과 거의 동일한 임산부 할인 제도를 운영해요. SRT 앱에서 임산부 인증 후 결제 시 자동 할인이 적용되며, 동반 1인 좌석 업그레이드도 가능해요.

  • 혜택 구조 — KTX와 동일
  • 인증 채널 — SRT 앱·홈페이지
  • 적용 노선 — 수서~동탄~부산·목포 등 전 노선

할인 혜택은 언제까지 받을 수 있나요?

임신 확인 후부터 출산 후 1년까지예요. 출산 후에도 1년간 유지되므로 산후조리 후 친정 방문 등에 활용할 수 있어요. 1년이 지나면 자동 종료돼요.

  • 시작 — 임산부 인증 등록 시점
  • 종료 — 출산 후 1년
  • 연장 — 다음 임신 확인 시 재인증 가능
