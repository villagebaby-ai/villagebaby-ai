산전 혈액검사는 임신 초기 기본 8종 → 중기·후기에 추가 검사로 진행. 빈혈·혈액형·풍진 면역·B형간염·매독·HIV·임신성 당뇨·임신중독증 모니터링이 핵심. 대부분 국민행복카드로 본인부담 거의 0원이에요.

임신 중기 — 추가 검사

  • 2차 기형아 검사 (쿼드) — 15~20주, AFP·hCG·uE3·인히빈A
  • 혈압·소변검사 — 임신중독증 모니터링 (매 정기검진)
  • 혈색소 재검 — 빈혈 진행 확인

임신 후기 — 추가 검사

  • 임신성 당뇨 검사 (50g·75g OGTT) — 24~28주
  • 빈혈 재검 — 분만 출혈 대비
  • GBS 검사 (B군 연쇄상구균) — 35~37주, 출산 시 항생제 결정
  • 응고 검사 — 분만 안전성

자주 묻는 질문

산전 혈액검사 언제 받나요?

① 임신 초기(8~12주): 기본 검사 8종 ② 임신 중기(15~20주): 2차 기형아 + 임신중독증 모니터링 ③ 임신 후기(24~28주): 임신성 당뇨 + 빈혈 재검 ④ 출산 직전: GBS 검사. 시기별로 다른 검사를 받아요.

비용은?

국민행복카드 임신바우처(100만원)로 대부분 커버돼요. 정기 산전검진 + 혈액검사 + 일반 초음파를 합쳐도 한도 안에서 충분.

임신성 당뇨 검사는 꼭 받나요?

네, 표준 검사예요. 임신 24~28주에 50g OGTT 1단계 검사. 양성이면 75g 2단계로 확진. 임신성 당뇨 약 5~10% 발생하므로 모든 임산부 권장.

B형간염 양성이면?

출생 후 신생아에게 면역글로불린 + 백신 접종으로 거의 100% 예방 가능해요. 임신 중에는 활동성 평가 + 모니터링.

Rh 부적합 검사는 뭐예요?

산모 Rh-, 태아 Rh+면 항체 형성 가능. 임신 28주에 RhoGAM 주사로 예방. 모르면 다음 임신에서 태아 빈혈·황달 위험.
