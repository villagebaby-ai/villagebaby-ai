산전검사 실제 케이스 시나리오 5가지
시기별 진단·청구 흐름

베이비빌리 베동 커뮤니티에 자주 등장하는 산전검사 패턴 5가지를 시뮬레이션으로 정리했어요. NIPT 양성 후 정상 확진, 정밀 초음파 이상 발견, OGTT 양성 등 시기별 흐름을 보여드려요.

케이스별 시나리오

케이스 1 — 38세 임산부 NIPT 양성 후 양수검사 정상

흐름: 임신 12주 NIPT 양성 → 1주 불안 → 16주 양수검사 → 정상 확진 → 정기 검진 계속

비용: NIPT 70만 + 양수검사 100만 = 본인 부담 약 170만원 (국민행복카드 100만원 차감 후 70만원)

보험 청구: 정상이라 청구 없음. 다만 검사 자체는 본인 부담

케이스 2 — 정밀 초음파에서 심장기형 발견

흐름: 임신 22주 정밀 초음파 → 심실중격결손 의심 → 태아 심초음파 → 확진 → 3차 병원 출산 계획

출생 후: 신생아 시기 경과 관찰, 2세에 시술

보험 청구: 출생 후 NICU 입원비 + 시술 시 수술비 특약 청구

케이스 3 — OGTT 양성으로 임신성 당뇨 진단

흐름: 임신 26주 50g OGTT 145 → 100g 확진 → 임신성 당뇨 진단 → 식이 조절

청구: 우체국 엄마보험 임신성 당뇨 진단비 3만원 + 민영 산모특약 100만원 = 103만원

케이스 4 — 양수 과다 진단 후 입원

흐름: 임신 32주 정기 초음파 AFI 28cm → 양수 과다 → 입원 5일 모니터링

청구: 산모 입원일당 3만원 × 5일 = 15만원 + 산모 입원 진단비 약 50만원

케이스 5 — 쌍둥이 임신 조산 후 NICU 입원

흐름: 임신 33주 조산 → 두 자녀 모두 NICU 입원 14일 → 정상 발달 후 퇴원

청구: NICU 입원비 50만원 × 14일 × 2명 + 정부 미숙아 의료비 지원 = 합산 약 2,000만원+

본인 케이스에 맞춰 무료로 정리해드릴게요.

위 케이스 외에도 본인 상황에 맞는 검사 흐름·보장 청구 가능 영역을 무료로 정리해드려요. 카톡으로 5분 안에 안내 가능해요.
